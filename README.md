## 1. Задание
Создать интерфейс программы, который должен обеспечивать выполнение операций:
 - Вычисление значения f(x) в точке;
 - Построение таблицы значений x → f(x) на интервале (с выводом в консоль и сохранением в файл);
 - Проверка монотонности функции (возрастание) на заданном интервале;
 - Поиск аргумента x по значению функции (f(x) ≈ Y);
 - Вычисление производной f’(x) в точке (численный метод).

## 3. Описание функций
**1. `double input_scanf(const char* message)`**
- **Назначение**: Безопасный ввод вещественного числа с клавиатуры.
- **Параметры**:
    - `message` - текст приглашения для ввода.
- **Особенности**: Блокирует ввод некорректных символов (букв), требуя повторного ввода.

**2. `double f(double x)`**
- **Назначение**: Вычисление значения кусочно-заданной функции.
- **Логика**:
    - При `x < -2`: Вычисляется `(cos(2x) - 1) / x^2`.
    - При `-2 <= x < 3`: Вычисляется `x * exp(-x/2) + (1 + x^3)^0.25`. Содержит проверку на отрицательное выражение под корнем.
    - При `x >= 3`: Вычисляется сложное полиномиально-логарифмическое выражение.
- **Возвращает**: Значение функции или `NAN`/`INF` в случае ошибки области определения.

**3. `void create_value_table(double a, double b, double step)`**
- **Назначение**: Вывод таблицы значений функции в консоль.
- **Параметры**:
    - `a`, `b` - границы отрезка.
    - `step` - шаг табуляции.
- **Особенности**: Обрабатывает специальные значения (`NAN`, `INF`) и выводит текстовое пояснение вместо чисел.

**4. `int save_table_to_file(double a, double b, double step, const char* filename)`**
- **Назначение**: Сохранение таблицы значений в текстовый файл.
- **Параметры**:
    - `filename` - имя файла для записи (например, "table.txt").
- **Возвращает**: `1` при успешной записи, `0` при ошибке открытия файла.

**5. `int check_increasing(double a, double b, double step)`**
- **Назначение**: Проверка функции на монотонное возрастание на отрезке.
- **Возвращает**:
    - `1`: Функция монотонно возрастает.
    - `0`: Функция не является возрастающей.
    - `-1`: На отрезке найдены точки разрыва или неопределенности.

**6. `int find_x_for_y(double y, double a, double b, double eps, double* result_x)`**
- **Назначение**: Численный поиск аргумента `x`, при котором `f(x) = y`.
- **Параметры**:
    - `y` - искомое значение функции.
    - `eps` - точность поиска (допустимая погрешность).
    - `result_x` - указатель для записи найденного корня.
- **Возвращает**: `1` (найдено), `0` (не найдено), `-1` (ошибка границ `a > b`).

**7. `double derivative(double x, double h)`**
- **Назначение**: Вычисление производной методом центральной разности.
- **Параметры**:
    - `x` - точка дифференцирования.
    - `h` - шаг приращения (`dx`).
- **Формула**: `(f(x + h) - f(x - h)) / (2 * h)`.

## 5. Формат данных

*   **Ввод:** Вещественные числа двойной точности (`double`).
*   **Вывод:**
    *   Результаты вычислений выводятся с настраиваемой пользователем точностью (по умолчанию до 15 значащих цифр `%g`).
    *   В таблице используется форматированный вывод с выравниванием столбцов.
    *   Математические ошибки отображаются как "НЕ ОПРЕДЕЛЕНО" или "БЕСКОНЕЧНО".
